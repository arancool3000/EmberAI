macOS installation

Install Ember

No Homebrew or pre-installed Python is required. Setup creates a private environment inside the Ember folder.

1 · Run

Double-click “Install Ember.command”

If macOS blocks it, click Done, open System Settings → Privacy & Security, and choose Open Anyway. Only do this for a copy downloaded from Ember’s official GitHub repository.

2 · Install

Return to the installer window

Press Return. Setup downloads a private Python 3.12 toolchain and Ember’s required packages, then opens Ember.

3 · Permissions

Grant permissions when needed

  • Screen Recording lets Ember understand the screen.
  • Accessibility allows mouse and keyboard control.
  • Input Monitoring enables global shortcuts and workflow recording.
  • Microphone is optional and used only for voice.

Next launches: double-click Ember.command in the main folder. It reuses the installed environment and can launch offline.

Troubleshooting

  • Do not choose “Move to Bin” in the Gatekeeper dialog; it deletes the launcher.
  • If setup cannot download packages, check the connection and rerun the installer.
  • Keep the Ember folder together; do not move individual source files out of it.
