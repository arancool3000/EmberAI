Windows installation

Install Ember

The installer creates a private environment inside the Ember folder. It does not replace your system Python or scatter packages around your computer.

1 · Run

Double-click “Install Ember.bat”

Windows may ask whether you want to run the file. Choose More info → Run anyway only if you downloaded Ember from its official GitHub repository.

2 · Wait

Let setup finish

The installer creates .venv, downloads the required packages, and opens Ember. If Python is missing and Windows Package Manager is available, it installs Python 3.12 for your user account.

3 · Permissions

Approve only what you use

  • Screen access: needed for visual computer control.
  • Accessibility/input control: needed for mouse and keyboard automation.
  • Microphone: needed only for voice features.

Next launches: double-click Ember.bat in the main folder. It reuses the private environment and does not reinstall everything.

Troubleshooting

  • If Python installation fails, install Python 3.12 from python.org, enable “Add Python to PATH,” and rerun the installer.
  • If Microsoft Defender prompts, verify the download source before allowing it.
  • Keep the Ember folder together; do not move individual source files out of it.
