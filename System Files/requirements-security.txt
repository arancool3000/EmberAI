# these — every engine degrades gracefully when its library is absent (see av_static.py). Install
# them to upgrade the built-in stdlib analysis to full-strength parsing:
#
#   pip install -r requirements-security.txt
#
# These pull in native builds; keep them OUT of the default requirements.txt so the base install
# stays lightweight and cross-platform (same pattern as requirements-voice.txt).
